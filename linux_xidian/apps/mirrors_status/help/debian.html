使用说明

以Squeeze为例, 编辑/etc/apt/sources.list文件, 在文件最前面添加以下条目(操作前请做好相应备份)

    deb ftp://linux.xidian.edu.cn/debian/ squeeze main non-free contrib
    deb ftp://linux.xidian.edu.cn/debian/ squeeze-proposed-updates main non-free contrib
    deb-src ftp://linux.xidian.edu.cn/debian/ squeeze main non-free contrib
    deb-src ftp://linux.xidian.edu.cn/debian/ squeeze-proposed-updates main non-free contrib

或者下载相应版本的sources.list, 覆盖/etc/apt/sources.list即可(操作前请做好相应备份)

  • squeeze
  • wheezy
